# ITERATION ONE: Developing the breathing app
- After improving on many of the issues from the first iteration, this second iteration is a lot closer to a more final and presentable product

## CHANGELOG/FEATURES ADDED
    - Added a selecting indicator on the BEGINNING SCREEN, as well as changing the button shape for the cycles
    - Re-sized and reformatted the BEGINNING SCREEN as a whole
    - Added a brand new in-between scene - the INFORMATION SCREEN, which provides the user a moment to collect themselves, as well as getting ready for going ahead with a peaceful session.
    - Added gradients in the BREATHING SCREEN, as well as DRASTICALLY improved the animations, to make them feel a lot more responsive and clean
## PMI
### PROS 
    - The animations and colour palette is feeling a lot more mellow, as opposed to the harsh variant from iteration 1
    - The added screen (the INFORMATION SCREEN) improves the UX, by giving users a moment to collect themselves, and start the breathing cycle when they actually feel ready
### CONS
    - The animations could still use further refinement - they are a little bit rigid at times
    - Perhaps making it clearer which button is currently selected in the BEGINNING SCREEN
