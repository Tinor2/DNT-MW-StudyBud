# DNT-MW-StudyBud
DNT MAJOR WORK 
A desk buddy to help you manage work, study, stress and life, in a minamilist and lofi setting
**SOFTWARE**
- A python app running on a ESP32 QUALIA and a TFT RGB666 Screen
- Utilizes the vectorio library for all visuals and animations
- Wifi connectivity
**HARDWARE**
- A button and a rotary encoder (Adafruit) for the user to interact with the product
- A speaker for notifications
- A TFT RGB666 screen
